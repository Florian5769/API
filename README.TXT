Prérequis 

Visual Studio Code
C# pour Visual Studio Code 
Kit de développement logiciel (SDK) .NET 7.0

Visual Studio Code lance le navigateur par défaut sur http://localhost:53895
Ajoutez /swagger à l’URL, https://localhost:53895/swagger.


Dans ASP.NET Core, les services tels que le contexte de base de données doivent être inscrits auprès du conteneur d’injection de dépendances. Le conteneur fournit le service aux contrôleurs.
